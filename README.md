# Node Task Tracker

Карточный таск трекер на express.

---

## Особенности

<ul>
<li>Фреймворк Express.js</li>
<li>TypeScript типизация</li>
<li>Авторизация express-auth</li>
<li>СУБД SQLite</li>
<li>Валидация входящих данных</li>
<li>Вынесение чувствительных переменных в среду</li>
</ul>

## Запуск и использование

Открыть терминал в корне проекта и ввести команду:
```bash
npm start
```

Сервер будет доступен по адресу
```
http://localhost:{порт}
```
